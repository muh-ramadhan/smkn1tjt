Toasts

Toasts are small messages that can appear on a success or on tapping any button. They are incredibly flexible and easy to use!

Top Toasts

Toasts that appear from the top. These will automatically hide in 3 seconds.

Toast Top Animated Toast Top Information Toast Top Confirmation Toast Top Warning

Bottom Toasts

Toasts that appear from the top. These will automatically hide in 3 seconds.

Toast Bottom Animated Toast Bottom Information Toast Bottom Confirmation Toast Bottom Warning

Tap or Programatic

We built notifications based on Bootstrap toast system, so you can trigger them via a tap or programmatically by simply using the Bootstrap classic code.

var toastID = document.getElementById('toast-1');
toastID = new bootstrap.Toast(toastID);
toastID.show();
